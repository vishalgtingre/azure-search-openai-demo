Here is an example code in Python to download the OHLC data for optimizing the portfolio using Modern Portfolio Theory with a budget of 1000 EUR for assets AAPL, MSFT, and TSLA from Yahoo Finance:\n\n```\nimport yfinance as yf\nimport numpy as np\n# Download OHLC data for assets AAPL, MSFT, TSLA from Yahoo Finance\ndata = yf.download(['AAPL', 'MSFT', 'TSLA'], start='2022-01-01')['Adj Close']\n\n# Set portfolio optimization parameters\nN = len(data.columns)\nbudget = 1000\niv = np.random.uniform(size=N)\niv /= iv.sum()\nprices = data.iloc[-1].values\nmean = data.pct_change().mean()\ncov = data.pct_change().cov()\nprint(data)\nprint(mean)\nprint(cov)\n# Set objective function\nset_objective(quicksum(quicksum(iv[i]*iv[j]*prices[i]*prices[j]*cov[i,j] for j in range(N)) for i in range(N)) - quicksum(iv[i]*prices[i]*mean[i] for i in range(N)))\n\n# Set budget constraint\nadd_constraint(quicksum(iv[i]*prices[i] for i in range(N)) <= budget)\n```\n\nNote: This code assumes that you have already installed the yfinance package
